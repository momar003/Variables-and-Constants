# Variables and Constants

## Objective
Learn how to declare and use initialized and uninitialized variables in NASM Assembly language.

## Flowchart

Start

↓

Load var1 (10)

↓

Load var2 (15)

↓

Add var1 + var2

↓

Store answer in result

↓

End

## Challenges

One challenge I encountered was learning how initialized and uninitialized variables work in Assembly language. I also had to understand how to use CPU registers to perform arithmetic operations. After testing the program with gdb, I verified that the result variable correctly stored the value 25.
